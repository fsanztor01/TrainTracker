    🏋️‍♂️ Inicio

    ¿Qué es TrainTracker?

    TrainTracker es una aplicación web para registrar y analizar tus entrenamientos, ayudándote a mejorar tu progreso y alcanzar tus metas.

    Navegación

    • Menú superior: Pestañas principales (Diario, Estadísticas, Perfil, etc.)
    • Navegación inferior: Acceso rápido a Diario, Estadísticas e Importar
    • Cambio de tema: Botón en la esquina superior derecha para cambiar entre modo oscuro/claro

    📝 Diario

    Crear una sesión

    • Pulsa "+ Nueva sesión"
    • Ingresa el nombre (ej: "Pecho y tríceps")
    • Selecciona la fecha
    • Pulsa "Guardar"

    Añadir ejercicios

    • Dentro de una sesión, pulsa "+ Ejercicio" en cualquier momento
    • No necesitas activar ningún modo de edición
    • Ingresa el nombre del ejercicio y pulsa "Guardar"

    Editar nombres de ejercicios

    • Haz clic directamente sobre el nombre del ejercicio para editarlo
    • Aparecerán botones "💾 Guardar" y "✕ Cancelar" para confirmar o descartar el cambio
    • También puedes presionar Enter para guardar o Escape para cancelar
    • Al cambiar el nombre, las estadísticas del nombre anterior no contarán (solo el nuevo nombre)

    Registrar sets

    Para cada ejercicio, puedes añadir:

    • KG: Peso levantado
    • Reps: Repeticiones realizadas
    • RIR: Repeticiones en reserva (esfuerzo percibido)

    Editar sesión

    • El botón ✏️ solo sirve para cambiar el nombre y la fecha de la sesión
    • Al pulsarlo, se abrirá un diálogo donde puedes modificar estos datos
    • No necesitas activar ningún modo para añadir ejercicios o editar nombres

    Notas de ejercicios

    • Pulsa "+ Nota" junto al nombre del ejercicio
    • Escribe tu nota (puedes usar saltos de línea)
    • Pulsa "Guardar"

    Copiar entrenamiento anterior

    Usa "📅 Copiar día actual de la semana pasada" para replicar el entrenamiento del mismo día de la semana anterior

    📊 Estadísticas

    Ver tu progreso

    • Ve a la pestaña "Estadísticas"
    • Selecciona la métrica que quieres analizar:
      • Volumen total
      • RIR promedio
      • Peso máximo
    • Filtra por ejercicio específico o "Todos los ejercicios"
    • Elige el período a analizar

    Interpretar los datos

    • Barras azules: Datos reales de cada semana
    • Línea verde: Tendencia/promedio
    • Tabla detallada: Estadísticas por ejercicio con porcentaje de progreso

    👤 Perfil

    Información personal

    • Ve a la pestaña "Perfil"
    • Completa tus datos:
      • Nombre y apellidos
      • Altura (cm)
      • Peso actual (kg)
      • % grasa corporal
    • Pulsa "Actualizar perfil"

    Medidas corporales

    • En la sección "Medidas Corporales" puedes registrar:
      • Brazos (cm)
      • Pecho (cm)
      • Cintura (cm)
      • Caderas (cm)
      • Piernas (cm)
      • Gemelos (cm)
    • Puedes registrar todas las medidas o solo algunas
    • Se guarda un historial con todas tus medidas por fecha
    • Pulsa "Guardar medidas" para registrar

    Foto de perfil

    Pulsa "Seleccionar archivo" para subir tu foto

    Personalizar colores

    • En la sección "Personalizar colores"
    • Elige tu color preferido para:
      • Modo oscuro
      • Modo claro
    • Haz clic en el color que te guste

    Calculadora de TMB y Calorías

    • En la sección "Calculadora de TMB y Calorías" puedes calcular:
      • TMB (Tasa Metabólica Basal): Las calorías que quemas en reposo
      • TDEE (Gasto Energético Total): Calorías totales según tu nivel de actividad
      • Calorías recomendadas: Según tu objetivo (mantener, perder grasa o ganar masa)
    • Selecciona tu sexo, edad y nivel de actividad
    • Elige tu objetivo (mantener peso, perder grasa o ganar masa muscular)
    • La calculadora usa automáticamente tu peso y altura del perfil
    • Pulsa "Calcular" para ver los resultados

    Notas rápidas

    Añade notas personales de hasta 200 caracteres. Se guardan con fecha y hora.

    🧩 Rutinas

    Crear rutina personalizada

    • Ve a la pestaña "Crear"
    • Ingresa un nombre para tu rutina
    • Pulsa "+ Añadir día" para cada día de entrenamiento
    • Para cada día:
      • Asigna un nombre (ej: "Día 1 - Pecho")
      • Añade ejercicios con "+ Ejercicio"
      • Configura sets para cada ejercicio
    • Pulsa "Guardar rutina"

    Usar plantillas predefinidas

    En la misma sección, encontrarás rutinas predefinidas:

    • 3 días
    • 4 días
    • 5 días
    • PPL 6 días

    Pulsa "Usar plantilla" para cargarla

    🎯 Objetivos

    Crear un objetivo

    • Ve a la pestaña "Objetivos"
    • Pulsa "+ Nuevo Objetivo"
    • Completa los datos:
      • Nombre del objetivo
      • Tipo de objetivo:
        • Aumentar peso (kg) - para ejercicios
        • Perder peso (kg) - basado en tu peso del perfil
        • Ganar peso (kg) - basado en tu peso del perfil
        • Aumentar volumen total
        • Aumentar repeticiones
        • Completar sesiones
        • Racha de días
        • Mejorar ejercicio específico
      • Meta numérica (para perder/ganar peso, es la cantidad a perder/ganar)
      • Ejercicio específico (si aplica)
      • Fecha límite (opcional)
    • Activa "Desglosar automáticamente en hitos" para crear metas intermedias
    • Pulsa "Guardar"

    Seguimiento de objetivos

    • Los objetivos muestran tu progreso actual
    • Los hitos se marcan automáticamente al alcanzarlos
    • Recibirás notificaciones al completar objetivos o hitos

    📥 Importar/Exportar

    Importar entrenamientos

    • Ve a la pestaña "Importar"
    • Selecciona la semana objetivo
    • Elige un archivo JSON válido
    • Revisa la vista previa
    • Pulsa "Importar"

    Usar plantillas rápidas

    Selecciona entre rutinas de 3, 4, 5 días o PPL. Se importarán automáticamente en la semana visible.

    Exportar datos

    Pulsa "Exportar sesiones" para descargar todos tus entrenamientos en formato JSON

    💡 Consejos Útiles

    Récords Personales (PRs)

    La aplicación detecta automáticamente tus récords. Verás una medalla 🏆 cuando superes tus marcas anteriores.

    1RM (Repetición Máxima)

    La aplicación calcula tu 1RM estimado basado en tus pesos y repeticiones. Se muestra junto a cada set completado.

    Modo Competitivo

    En tu perfil, encontrarás:

    • Racha actual: Días consecutivos entrenando
    • Meta semanal: Progreso hacia tu objetivo semanal
    • Logros desbloqueados: Reconocimientos por tus logros

    Atajos

    • Doble clic: Edita rápidamente nombres de ejercicios
    • Tab: Navega entre campos de entrada
    • Escape: Cancela edición o cierra diálogos

    ❓ Preguntas Frecuentes

    ¿Se guardan automáticamente mis datos?

    Sí, todos tus cambios se guardan localmente en tu navegador.

    ¿Puedo usar la aplicación sin conexión a internet?

    Sí, una vez cargada, la aplicación funciona completamente offline.

    ¿Puedo exportar mis datos?

    Sí, usa la función "Exportar sesiones" en la pestaña Importar.

    ¿Cómo puedo editar un ejercicio ya creado?

    Haz clic directamente sobre el nombre del ejercicio para editarlo. Aparecerán botones de guardar/cancelar para confirmar el cambio.

    ¿Puedo eliminar ejercicios?

    Sí, puedes eliminar ejercicios en cualquier momento usando el botón ✕ junto al nombre del ejercicio. Aparecerá un diálogo de confirmación.

    ¿Cómo funcionan las estadísticas de progreso?

    Las estadísticas comparan el período actual con el período anterior del mismo tamaño. Si no hay datos del período anterior, se compara con el primer registro de ese ejercicio para calcular el progreso.
